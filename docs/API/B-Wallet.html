Перевести средства между своими счетами + конвертация

Выполняет перевод средств между своими счетами, такими как кошельки и карты, в том числе позволяет выполнить конвертацию электронных рублей на цифровые и наоборот.

Authorizations:
BearerAuth
header Parameters
Idempotency-Key
string

Уникальный идентификатор для обеспечения идемпотентности запроса

Request Body schema: application/json
required
fromAccount
required
string

Уникальный идентификатор счета, с которого будет списана сумма (publicKey или cardId)

toAccount
required
string

Уникальный идентификатор счета, на который будут зачислены средства (publicKey или cardId)

amount
required
number <float>

Сумма для перевода

Responses

Основной сервер

https://api.blockchain-wallet.com/payments/betweenaccounts/transfer

Request samples

  • Payload
Content type
application/json

  • "fromAccount": "wallet_12345",
  • "toAccount": "card_67890",
  • "amount": 2000.75
}

Response samples

  • 200
  • 400
  • 404
  • 500

Operations

История операций

Просмотр истории операций с курсорной пагинацией

Получает историю операций для всех кошельков и карт клиента, с возможностью фильтрации по категории и дате, используя курсорную пагинацию.

Authorizations:
BearerAuth
query Parameters
startDate
string <date>
Example: startDate=2024-01-01

Начальная дата для фильтрации операций

endDate
string <date>
Example: endDate=2024-12-31

Конечная дата для фильтрации операций

category
string
Enum: "Переводы" "Развлечения" "Супермаркеты" "Одежда и обувь" "НКО" "Остальное"
Example: category=shopping

Категория операций для фильтрации

cursor
string
Example: cursor=tx12345

Курсор для получения следующей страницы данных. Пустое значение означает первую страницу.

limit
required
integer
Example: limit=10

Количество записей, которые нужно вернуть на одной странице.

Responses

Основной сервер

https://api.blockchain-wallet.com/main/operations

Response samples

  • 200
  • 400
  • 404
  • 500
Content type
application/json
{
  • "total": 150,
  • "operations": [
    • {
      • "transactionId": "tx12345",
      • "date": "2024-11-10T12:00:00Z",
      • "amount": 1500.5,
      • "category": "shopping",
      • "fromAccount": "wallet_12345",
      • "toAccount": "card_67890"
      }
    ],
  • "nextCursor": "tx67890"
}
